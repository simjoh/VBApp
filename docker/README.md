#### Starta allt i docker
- Öppna terminal i mappen i projektets root
    - docker-compose up
  #### Starta och tvinga ombyggnad
    - docker-compose up --force-recreate
- #### Stoppa alla tjänster
- Öppna terminal i mappen i projektets root
  - docker-compose down
  #### Stoppa och tabort allokerade volymer
  - docker-compose down -v
